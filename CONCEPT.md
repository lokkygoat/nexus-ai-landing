## Интерактивные фичи

### Курсор
- [ ] **Cursor follower** — круг с задержкой следует за мышкой
- [ ] **Cursor spotlight** — тёмный фон, светлый круг вокруг курсора
- [ ] **Cursor morphing** — меняет форму на тексте/кнопках

### Кнопки
- [ ] **Magnetic buttons** — кнопка тянется к курсору на ~20px
- [ ] **Ripple effect** — волна от точки нажатия
- [ ] **Liquid fill** — при наведении заполняется жидкостью снизу
- [ ] **Glow pulse** — пульсирующее свечение на CTA кнопках

### Текст
- [ ] **Scramble text** — буквы хаотично меняются, потом складываются (Matrix эффект)
- [ ] **Typewriter** — печатается: "автоматизируем продажи / поддержку / маркетинг"
- [ ] **Reveal on scroll** — текст открывается маской построчно
- [ ] **Glitch эффект** — на главном заголовке Hero

### Анимации
- [ ] **Нейросеть** — точки соединены линиями, реагируют на мышку (Canvas)
- [ ] **Floating blobs** — живые градиентные шары на фоне
- [ ] **Parallax** — слои двигаются с разной скоростью при скролле
- [ ] **Counting numbers** — цифры крутятся при появлении секции Stats
- [ ] **Stack cards scroll** — карточки услуг уходят вверх стопкой
- [ ] **Infinite marquee** — бегущая строка технологий, замедляется при наведении

### Wow-блоки
- [ ] **ROI Калькулятор** — 3 ползунка → анимированный результат в ₽
- [ ] **Progress bar** — тонкая линия сверху, показывает скролл страницы
- [ ] **Easter egg** — Konami Code (↑↑↓↓←→BA) → что-то взрывается
- [ ] **Page transitions** — noise/static при смене секций

## Технологии

- **HTML5 + CSS3 + Vanilla JS** (без фреймворков — быстрая загрузка)

- **Canvas API** — нейросеть и particle effects
- **CSS Custom Properties** — для анимаций и тем

## ROI Калькулятор — детали

```
Ползунок 1: Количество сотрудников (1–100)
Ползунок 2: Часов рутины в день (0.5–8)
Ползунок 3: Средняя зарплата (₽30k–₽300k)

Формула:
  Потери в месяц = сотрудники × часы × (зарплата / 160 рабочих часов) × 22 дня
  Экономия с AI = потери × 0.85 (85% рутины берёт AI)
  Окупаемость = стоимость внедрения / экономия в месяц

Вывод:
  🔴 Вы теряете X ₽/мес
  🟢 С NEXUS AI экономите Y ₽/мес
  ⚡ Окупаемость за Z дней
```
